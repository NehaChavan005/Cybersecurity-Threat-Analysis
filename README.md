# 🔐 Cybersecurity Threat Analysis Dashboard (Power BI)
A Cybersecurity Threat Analysis dashboard in Power BI visualizes and monitors security data to identify potential threats, vulnerabilities, and attack patterns. It tracks metrics like attack types, frequency, affected systems, and response times, helping organizations detect risks early and improve their overall security posture.


📁 Project Overview
This project is a Power BI dashboard (.pbix) designed to analyze and visualize cybersecurity data. It helps identify potential threats, monitor attack patterns, and assess system vulnerabilities through interactive visuals.
The dashboard enables organizations to enhance security by providing clear insights into cyber risks and incidents.

🎯 Objectives
1. Detect and analyze cybersecurity threats
2. Monitor attack frequency and patterns
3. Identify vulnerable systems and areas
4. Track incident response performance
5. Support proactive security decision-making

📦 Files Included
________.pbix – Main dashboard file

🛠️ Tools & Technologies
1. Power BI Desktop
2. Power Query (Data Cleaning & Transformation)
3. DAX (Data Analysis Expressions)

📊 Dashboard Features
🔹 1. Overview
      -Total number of attacks
      -Active vs resolved threats
      -High-level security KPIs
🔹 2. Threat Analysis
      -Types of attacks (Malware, Phishing, DDoS, etc.)
      -Attack trends over time
      -Severity levels
🔹 3. Vulnerability Analysis
      -Affected systems
      -Risk levels by department or region
      -Most targeted assets
🔹 4. Incident Response
      -Response time analysis
      -Resolution rates
      -Incident handling performance
🔹 5. Interactive Filters
      -Date range slicer
      -Attack type filter
      -Region/system filters

📈 Key Metrics
1. Total Threats Detected
2. High/Medium/Low Severity Incidents
3. Average Response Time
4. Resolution Rate (%)
5. Most Frequent Attack Type
